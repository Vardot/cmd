# Vardot Commands

### Setup Behat UI module and configs for DevShop projects.
1. Go to the project directory:
```
cd /path/to/project
```
2. Run the interactive command: 
```
bash <(wget -O - https://raw.githubusercontent.com/Vardot/cmd/master/behat_ui_devshop.sh)
```
